作っているもの

rshogi

SH11235/rshogi

Rust 製 USI 将棋エンジン。ビットボード、Alpha-Beta 探索、NNUE 評価関数、Lazy SMP。教師生成・定跡・分析用の CLI ツール 58 本を同梱。

対局インフラ

rshogi/crates/rshogi-csa-*

CSA プロトコルのクライアント・サーバー。本番は Cloudflare Workers + R2 で稼働し、floodgate にも接続。

ラム将棋

ramu-shogi.sh11235.com

Web・デスクトップ将棋アプリ。rshogi を WASM で内蔵。CSA 対局の live 観戦ビューアー付き。

tatara

SH11235/tatara

NNUE 評価関数の学習ツール。CUDA カーネルを Rust から生成。bullet-shogi から発展。

nnue-lab

nnue-lab.sh11235.com/t/sh11235

学習実験の記録・可視化 Web アプリ。loss カーブの重ね描き比較、lineage 連結、NNUE・エンジンバイナリの管理。

bullet-shogi

SH11235/bullet-shogi

NNUE 学習ライブラリ bullet の将棋特化 fork。tatara の前身で、現在は tatara への移行を推奨。
